Что такое HTML?

HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. Он определяет структуру и содержание веб-страницы.

Основные понятия:

  • Теги - основные строительные блоки HTML, заключенные в угловые скобки (<>)
  • Элементы - теги с содержимым (например, <p>Текст</p>)
  • Атрибуты - дополнительные параметры элементов (например, <a href="...">)

Структура HTML-документа

<!DOCTYPE html>
<html lang="ru">
<head>
  <meta charset="UTF-8">
  <title>Заголовок страницы</title>
</head>
<body>
  <h1>Мой первый заголовок</h1>
  <p>Мой первый абзац.</p>
</body>
</html>
Примечание: Все HTML-документы должны начинаться с декларации <!DOCTYPE html>.

Основные HTML-теги

Тег Описание Пример
<h1>-<h6> Заголовки разного уровня <h1>Главный заголовок</h1>
<p> Абзац <p>Это абзац текста.</p>
<a> Ссылка <a href="https://example.com">Ссылка</a>
<img> Изображение <img src="image.jpg" alt="Описание">
<ul>, <ol>, <li> Списки <ul><li>Пункт 1</li></ul>
<div> Блочный контейнер <div>Блок содержимого</div>

HTML-атрибуты

Атрибуты предоставляют дополнительную информацию об элементах:

  • href - указывает URL для ссылок
  • src - указывает путь к изображению
  • alt - альтернативный текст для изображений
  • class - определяет класс для CSS
  • id - уникальный идентификатор элемента

Формы в HTML

Формы используются для сбора пользовательского ввода:

<form action="/submit" method="post">
  <label for="name">Имя:</label>
  <input type="text" id="name" name="name">
  
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  
  <input type="submit" value="Отправить">
</form>

Проверьте свои знания HTML

1. Какой тег используется для создания заголовка самого высокого уровня?

<h1>
<head>
<header>
Тег <h1> используется для заголовков самого высокого уровня (Heading 1). Теги <head> и <header> имеют другое назначение.

2. Какой атрибут обязателен для тега <img>?

href
src
link
Атрибут src (source) обязателен для тега <img> и указывает путь к изображению. Атрибут href используется для ссылок (<a>).

3. Какой тег используется для создания абзаца?

<br>
<p>
<div>
Тег <p> создает абзац. <br> - перенос строки, <div> - блочный контейнер.

4. Какой тег должен быть первым в HTML-документе?

<!DOCTYPE html>
<html>
<head>
Декларация <!DOCTYPE html> должна быть первой в документе и указывает браузеру, что это HTML5 документ.

5. Какой атрибут используется для создания гиперссылки?

src
href
link
Атрибут href (Hypertext Reference) используется в теге <a> для создания гиперссылок.
